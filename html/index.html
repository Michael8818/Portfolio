Automatic Card Game Simulator (Windows Desktop)

I led development of a fully automated digital card game simulator built in Unity for Windows desktop. Players can build or import decks, compete in server-authoritative multiplayer matches using Photon, or play against a heuristic-driven AI.

The deterministic rules engine automatically resolves turn phases, stack/priority interactions, triggered effects, interrupts, and illegal move validation through a structured state machine. Game state synchronization was managed through coroutine-based flow control to ensure consistent client alignment.

The deck builder supports filtering, validation, and import/export across a pool of ~2,500 cards. Complex effect execution required performance safeguards to maintain stability.

Released publicly with matchmaking and live patch support, the project was developed as Lead Developer within a six-person team.

Medical Room Planner (Windows Desktop)

I developed a Windows desktop medical room planning application built in Unity that enables users to import DXF CAD files and design, manipulate, and review clinical spaces in real time. The system was architected to support complex hospital layouts, including full wings with 3,000+ walls and hundreds of medical equipment models.

I built a custom DXF import pipeline from scratch, including unit conversion logic that dynamically adapts to user-defined measurement settings. The application features snapping, grid alignment, collision handling, and constraint-based object movement modeled after real-world spatial rules. A modular architecture supports rapid integration of new equipment models and feature expansion.

The tool includes multiple review modes—top-down, orthographic side view, 3D perspective, and first-person—along with equipment lighting controls for realistic spatial evaluation. Layout data is persisted locally using a custom JSON format.

Extensive optimization work ensured stable performance under heavy geometry loads. The application increased delivery speed by 40% and improved client confidence by 60%.

Key Cutting Training Game (WebGL Browser)

I developed a browser-based key cutting training game in Unity, deployed via WebGL, combining mechanical simulation with interactive mini-games to reinforce learning. The core experience features a fully interactive 3D key cutting machine with physics-based constraints, clamp states, tracer alignment, and precision-based scoring tied to timing and sequence accuracy. Failure states, percentage feedback, and contextual hints guide player improvement.

Key data and scenarios were built using ScriptableObjects for scalable content management. Reinforcement mini-games procedurally spawn keys and knowledge challenges, including a custom driving game that tests recognition under pressure.

Optimized specifically for WebGL performance limitations, the application supports 30-minute client training sessions and was developed as part of a two-person team.
